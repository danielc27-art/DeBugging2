ToDo List

    How This Page Should Work:

    • Input Field (input#taskInput): User types a task description here. Should be cleared after adding a task.

      • Get the text from the input field
      • Create a new list item with the task text
      • Add a "Remove" button to each task item
      • Append the new item to the task list
      • Clear the input field
    • Task List (ul#taskList): Container that holds all task items. Starts empty.
    • Remove Button (on each task): When clicked, should:
      • Remove that specific task item from the list
