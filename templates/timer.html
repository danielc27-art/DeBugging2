How This Page Should Work:

  • Timer Display (div#timer): Shows elapsed time in MM:SS format. Starts at 00:00.
  • Start Button: When clicked, should:
    • Begin counting up from the current time
    • Update the display every second (1000ms)
    • Continue until Stop or Reset is clicked
  • Stop Button: When clicked, should:
    • Pause the timer (stop the interval)
    • Keep the current time displayed
  • Reset Button: When clicked, should:
    • Stop the timer if it's running
    • Reset the time back to 00:00
