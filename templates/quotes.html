Random Quote Generator

Click the button to get a random quote!

How This Page Should Work:

  • Quote Display (div#quoteDisplay): Container that shows the quote text and author. Initially shows a placeholder message.
  • Quote Text (div#quoteText): Displays the quote message.
  • Quote Author (div#quoteAuthor): Displays the author of the quote.
  • Generate Button: When clicked, should:
    • Select a random quote from a predefined array of quotes
    • Update the quote text element with the selected quote
    • Update the author element with the quote's author
  • Quotes Array: Contains multiple quote objects, each with a "text" and "author" property. The function should randomly pick one each time.
